# MyNPC

一個高級的打人NPC 擁有超強的運作能力，並且不斷的更新，打破你對NPC的想法。基于GPL协议开源
-注意，当前仅支持php5.
指令
- /npc 重新reload個新的npc
- /new 添加新的NPC

目前更能
- 可自定義NAMETAG
- 可自定義血量/最大血量
- 可自定義移動速度
- 可自定義移動範圍
- 可自定義更多…

未來添加
- 自定義手持
- 自定義皮膚
- 普通NPC的移動（不會攻擊玩家)
